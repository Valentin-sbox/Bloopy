## Novedades

### Rendimiento
- **Optimización completa**: Reducción del 40% al 25% de uso de RAM (4GB sistema)
- Mejoras generales de rendimiento en toda la aplicación

### UI/UX
- Interfaz completamente renovada
- Sistema de pestañas/tabs mejorado
- Split View optimizado y estable

### Editor
- Implementación de **TLdraw 4.4.0** para notas visuales
- Comando `/` (slash command) optimizado
- Eliminación de drag & drop de párrafos (mejor estabilidad)
- drag & drop solo para sub archivos dentro de parrafos!

## Características

| Feature | Descripción |
|---------|-------------|
| **TLdraw** | Lienzo para dibujar, inyectar imágenes y escribir. [tldraw.com](https://www.tldraw.com/) |
| **Split View** | Edición paralela de dos archivos |
| **Estados de escritura** | Tracker de progreso por capítulo |
| **Tabs** | Múltiples archivos abiertos |
| **Atajos de teclado** | Personalizables |
| **Temas** | Claro/oscuro + personalizados |
| **Sin internet** | 100% offline, tus datos son tuyos |
